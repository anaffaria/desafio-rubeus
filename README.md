# 🔎 Teste Técnico — Análise de Qualidade e Reporte de Bugs

## 📌 Sobre o Projeto

Este repositório contém o resultado de um teste técnico cujo objetivo foi avaliar a qualidade de duas páginas web, com foco em:

- Identificação de bugs  
- Detalhamento de inconsistências  
- Problemas de usabilidade  
- Documentação clara e organizada  

A análise foi documentada e acompanhada em um **Kanban no Notion**.
